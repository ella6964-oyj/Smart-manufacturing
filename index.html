📋 Nelson's Rule 8가지 이상판정 규칙
Rule 1
1개 이상의 관측치가 ±3σ를 벗어남
Rule 2
연속 9개 이상의 관측치가 중심선 위 또는 아래에 존재 (bias)
Rule 3
연속 6개 이상의 관측치가 계속 증가 또는 감소 (추세)
Rule 4
연속 14개 이상의 관측치가 번갈아 증가/감소 (진동)
Rule 5
연속 3개 중 2개 이상이 ±2σ 외부에 같은 방향으로 존재
Rule 6
연속 5개 중 4개 이상이 ±1σ 외부에 같은 방향으로 존재
Rule 7
연속 15개 이상의 관측치가 ±1σ 범위 내에 존재 (과소산포)
Rule 8
연속 8개의 관측치가 모두 ±1σ 밖에 위치
📐 관리도 선택 기준
데이터 유형부분군 크기관리도
계량형n = 1I-MR (개별-이동범위)
계량형n = 2~9Xbar-R (평균-범위)
계량형n ≥ 10Xbar-S (평균-표준편차)
계수형 (불량수)일정NP 관리도
계수형 (불량률)가변P 관리도
계수형 (결함수)일정C 관리도
계수형 (결함률)가변U 관리도
⚙️ σ 계산 방법

σ_within (단기):

  • n=1: σ = MR̄ / d₂(w)
  • n≥2 (범위): σ = R̄ / d₂(n)
  • n≥2 (표준편차): σ = s̄ / c₄(n)

σ_overall (장기):

  • σ = s / c₄(n) (전체 표준편차)
